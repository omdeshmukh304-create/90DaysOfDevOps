# Git Commands Reference

## Setup & Config

git --version  
Shows installed Git version  
Example: git --version

git config --global user.name "Your Name"  
Sets username for commits  
Example: git config --global user.name "Om Deshmukh"

git config --global user.email "your@email.com"  
Sets email for commits  
Example: git config --global user.email "omdeshmukh304@gmail.com"

---

## Basic Workflow

git init  
Initializes a new repository  
Example: git init

git status  
Shows current state of repository  
Example: git status

git add <file>  
Moves file to staging area  
Example: git add git-commands.md

git commit -m "message"  
Saves snapshot permanently  
Example: git commit -m "Add commands file"

---

## Viewing Changes

git diff  
Shows changes made but not committed  
Example: git diff

git log  
Shows commit history  
Example: git log

git log --oneline  
Shows compact history  
Example: git log --oneline
